Generator¶

class acoular.base.Generator¶

Bases: HasPrivateTraits

Interface for any generating signal processing block.

It provides a common interface for all classes, which generate an output via the generator result() in block-wise manner. It has a common set of traits that are used by all classes that implement this interface. This includes the sampling frequency of the signal (sample_freq) and the number of samples (numsamples). A private trait digest is used to store the internal identifier of the object, which is a hash of the object’s attributes. This is used to check if the object’s internal state has changed.

sample_freq = Float(1.0, desc='sampling frequency')¶

Sampling frequency of the signal, defaults to 1.0

numsamples = CLong¶

Number of signal samples
