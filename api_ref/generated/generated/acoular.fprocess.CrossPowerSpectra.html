CrossPowerSpectra¶

class acoular.fprocess.CrossPowerSpectra¶

Bases: AutoPowerSpectra

Calculates the complex-valued auto- and cross-power spectra.

Receives the complex-valued spectra from the source and returns the cross-spectral matrix (CSM) in a flattened representation (i.e. the auto- and cross-power spectra are concatenated along the last axis). If calc_mode is ‘full’, the full CSM is calculated, if ‘upper’, only the upper triangle is calculated.

source = Trait(SpectraGenerator)¶

Data source; SpectraGenerator or derived object.

precision = Trait('complex128', 'complex64', desc='precision of the fft')¶

The floating-number-precision of entries of csm, eigenvalues and eigenvectors, corresponding to numpy dtypes. Default is 64 bit.

numchannels = Property(depends_on='source.numchannels')¶

Number of channels in output. If calc_mode is ‘full’, then numchannels is \(n^2\), where \(n\) is the number of channels in the input. If calc_mode is ‘upper’, then numchannels is \(n + n(n-1)/2\).

result(num=1)¶

Python generator that yields the output block-wise.

Parameters:
numinteger

This parameter defines the size of the blocks to be yielded (i.e. the number of samples per block).

Yields:
numpy.ndarray

Yields blocks of shape (num, numchannels*numfreq).
