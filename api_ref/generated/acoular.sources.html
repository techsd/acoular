PointSource

Class to define a fixed point source with an arbitrary signal.

PointSourceDipole

Class to define a fixed point source with an arbitrary signal and dipole characteristics via superposition of two nearby inversely phased monopoles.

Class to define a fixed Spherical Harmonic Source with an arbitrary signal.

LineSource

Class to define a point source with an arbitrary signal moving along a given trajectory.
